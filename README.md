# EduShare

## About the Project
This project provides a platform where university students can share their study resources (books, notes, electronic devices, etc.) either for a fee or for free. Students can earn extra income by selling resources they no longer need, while others can easily access these resources. 

## Features
- **User Registration and Authentication**: Secure sign-up and login system using Firebase Authentication.
- **Resource Sharing**: Allows users to create and share listings for notes, books, or devices.
- **Paid and Free Options**: Users can publish resources as either paid or free.
- **File Storage**: Upload and store resource files with Firebase Storage.

## Technologies Used
- **Frontend**: Flutter (Developed with Dart)
- **Backend**: Firebase (Firestore database, Firebase Authentication, and Firebase Storage)
